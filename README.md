# TCG Space

¡Bienvenido a **TCG Space**! Un emocionante juego de cartas con temática espacial de la Game Jam 2026.

## 🚀 Cómo Jugar (Sin instalación de Python)

No necesitas instalar Python, ni librerías, ni usar la terminal. Sigue estos sencillos pasos:

1. Ve a la pestaña **[Releases](../../releases)** (o "Lanzamientos") que está en el menú lateral derecho de este repositorio en GitHub.
2. Encuentra la versión más reciente del juego.
3. Descarga el archivo `TCG_Space_Windows.zip`.
4. Descomprime el archivo `.zip` en tu ordenador donde prefieras (por ejemplo en tu Escritorio).
5. Entra en la carpeta descomprimida y ejecuta el archivo **`TCG Space.exe`** haciendo doble clic.
6. ¡A jugar!

---

### Notas de Privacidad y Seguridad
El ejecutable del juego (`.exe`) se genera automáticamente a través de los servidores de GitHub Actions. Esto significa que **en ningún momento** el juego incluye información privada del creador o rutas locales de su ordenador, garantizando la privacidad absoluta tanto del creador como de los jugadores.

---

### Detalles Técnicos (Para Desarrolladores)
Si quieres clonar este repositorio y ver/modificar el código fuente:

1. Clona el repositorio: `git clone https://github.com/jordiarmbar/GameJam2026.git`
2. Navega a la carpeta descargada: `cd GameJam2026`
3. Asegúrate de tener Python 3 instalado.
4. Instala las dependencias: `pip install -r requirements.txt`
5. Ejecuta el juego base: `python main.py`
